社会各界如有依法合规，资质齐全的防护物资(耗材)， 恳请伸出援手！

!!! note "防护物资（耗材）清单"

    N95口罩、医用外科口罩、一次性医用口罩、防护帽、一次性医用防护服、手术衣、防护眼镜/眼罩/护目镜、医用一次性乳胶手套

!!! note "部分防护物资（耗材）需符合或高于下列国家标准"

    + 医用防护口罩 GB 19083-2010
    + 医用外科口罩 YY 0469-2010
    + 防护服 GB 19082-2003 《医用一次性防护衣技术要求》
